# Jain Silver Plaza - Web Application

Web version of Jain Silver Plaza built with React and Vite, ready to deploy on Vercel.

## Features

- User registration with document upload
- OTP verification
- Sign in (email/phone)
- Real-time silver rates
- Admin dashboard
- Responsive design

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Vercel will automatically detect Vite configuration
4. Deploy!

Or use Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Environment Variables

No environment variables needed - uses the same backend API as mobile app.
